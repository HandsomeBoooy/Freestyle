Bosch BMI088高性能IMU（用于无人机/机器人）

制造商：Bosch Sensortec

Bosch BMI088高性能惯性测量单元 (IMU) 专门设计用于无人机和机器人应用。该款6轴传感器在3 x 4.5 x 0.95mm³小尺寸LGA封装中集成了16位三轴陀螺仪和16位三轴加速度计。该单元采用经过汽车验证的陀螺仪技术和全新的低TCO加速度计设计。BMI088经过汽车验证的陀螺仪具有无与伦比的偏置稳定性（低于2°/h）和低于15mdps/K的低温度系数偏移 (TCO)。其他特性包括0.2mg/K的低TCO，以及在±24g的极宽测量范围内具有仅为230μg/√Hz的低频谱噪声。借助BMI088器件，即使在自然的高振动环境中也能更轻松地进行精确转向，因此改善了飞行和导航体验。
